NutriAI – AI Powered Nutrition Analyzer
Project Description

NutriAI is an AI-powered nutrition analysis web application built using Python, Streamlit, and Google Gemini AI.
The application allows users to enter food items and instantly receive detailed nutritional information generated by an AI model.

The system uses Google's Generative AI model to analyze food inputs and provide structured nutrition data such as calories, protein, fats, vitamins, and minerals.

Features

- Interactive web interface using Streamlit
- Accepts multiple food items as input
- Uses Google Gemini AI for intelligent nutrition analysis
- Displays detailed nutrition information instantly
- Simple, lightweight, and easy to run locally

Technologies Used

- Python
- Streamlit
- Google Generative AI (Gemini API)
  Project Structure

app.py – Main Streamlit application containing UI and logic
requirements.txt – Required Python libraries
  Installation & Setup

Step 1: Clone the repository or download files

Step 2: Install dependencies

pip install -r requirements.txt

Step 3: Add your Gemini API key

Open app.py and replace the API key with your own key:
genai.configure(api_key="YOUR_API_KEY")

Step 4: Run the application

streamlit run app.py

Step 5: Open in browser

The app will automatically open at:
http://localhost:8501

How It Works

1. User enters food items separated by commas
2. The app sends the input to Gemini AI
3. AI generates detailed nutrition information
4. Results are displayed on the webpage

Example Input

Rice, Egg, Milk, Apple
